Project Overview: Polyglot TTS

I developed a Python-based automation tool that transforms multilingual text into high-quality speech assets. Using the gTTS library, the script processes complex data structures to generate organized audio directories.

Core Features

  • Automated Directory Mapping: Automatically creates unique folders for each phrase ID.
  • Six-Language Support: Simultaneous generation for EN, SK, TR, HU, IT, and RO.
  • Efficient File Handling: Includes logic to skip existing files to save processing time.

Technical Stack

This project demonstrates my proficiency in Python 3, file system operations (os and path modules), and integrating third-party APIs for generative media.
